# QuASIM Notebook Collection

The notebooks packaged with the QuASIM Full Package Builder provide reproducible demonstrations
of validation workflows, Monte Carlo campaigns, and hardware configuration studies. These notebooks
are curated snapshots designed for offline review by auditors and engineering teams.

## Contents

- `hardware/` — PTAQ qubit lifecycle notebooks and acceptance test analyses.
- `compliance/` — (reserved) Documentation walkthroughs for certification bodies.
- `analysis/` — (reserved) Market and valuation notebooks aligned with investor reporting.

All notebooks are executed in a controlled environment specified by `docker/notebook.Dockerfile`.
