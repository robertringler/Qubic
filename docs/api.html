Quick Start

Bash 
pip install quasim
quasim --version

Initialize a quantum simulator with GPU acceleration.

from quasim import QuantumSimulator

sim = QuantumSimulator(
    qubits=32,
    backend='cuQuantum',
    gpus=[0, 1, 2, 3],
    precision='fp32'
)

Build quantum circuits with gates and measurements.

from quasim.circuits import QuantumCircuit

circuit = QuantumCircuit(4)
circuit.h(0)
circuit.cx(0, 1)
circuit.cx(1, 2)
circuit.cx(2, 3)
circuit.measure_all()

Run circuits and analyze results.

result = sim.run(circuit, shots=1024)
print(f"Fidelity: {result.fidelity:.4f}")
print(f"Counts: {result.counts}")
